#  Simple Weather App 

A React project that displays weather data with multi-language support.

## 🚀 Features
- 🌍 Multi-language support using i18next
- ⏰ Date formatting using dayjs
- 🌦️ Weather data from API
- 📱 Responsive design

## 🛠️ Technologies Used
- React
- i18next
- dayjs
- API (Weather API)

## 📦 Installation

```bash
npm install
npm start
